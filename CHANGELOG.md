# Changelog

## 1.2.0 - 2026-07-24

- 新增“用 CC 打开”，可在 iTerm2 中进入对应目录并启动 Claude Code。

## 1.1.2 - 2026-07-22

- 将实际 App 包名和发布文件名统一为 `MagicMenu`，并自动清理构建副本，避免启动台出现重复应用。

## 1.1.1 - 2026-07-22

- 将面向用户的 App 名称简化为 `MagicMenu`，保留现有包标识和内部文件名以兼容已有授权与配置。

## 1.1.0 - 2026-07-21

- 重新设计 App 图标，简化为文件夹、菜单卡片和魔法星芒，提升小尺寸辨识度。
- 新建文件改用原子不覆盖写入，消除并发操作下的同名文件覆盖窗口。
- 拒绝缺失 `action` 的扩展请求，避免将损坏请求默认解释为新建文件。
- 增强 URL 参数解析，重复查询键不再导致崩溃。
- 统一 App 和 Finder 扩展的版本配置，新增可复现的 Release 打包流程。
